📝 To-Do List Web App

A simple and interactive To-Do List Web Application that helps users manage their daily tasks efficiently.
This project allows users to add tasks, mark them as completed, edit tasks, and delete tasks.

This project was developed as Level 2 – Task 3 from my Oasis Infobyte Internship Journey.

---

🚀 Features

✔ Add new tasks
✔ Mark tasks as completed
✔ View pending tasks
✔ View completed tasks
✔ Edit tasks
✔ Delete tasks
✔ Simple and responsive user interface
